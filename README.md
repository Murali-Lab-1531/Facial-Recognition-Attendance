# Facial Recognition Attendance System

## 📌 Overview
A smart attendance management system that leverages facial recognition technology to automatically identify individuals and mark attendance. This eliminates manual roll calls, reduces proxy attendance, and ensures accurate record-keeping.

## 🚀 Features
- Real-time face detection and recognition using AI models (FaceNet, MTCNN, OpenCV).
- Secure database integration for storing attendance logs.
- Role-based access (Admin, Faculty, Student).
- Dashboard with analytics (daily, monthly, department-wise attendance).
- Export attendance reports in CSV/Excel format.

## 🛠️ Tech Stack
- **Backend**: Python (Flask/Django)
- **Frontend**: HTML, CSS, Bootstrap
- **Database**: MySQL/PostgreSQL
- **AI Models**: CNN-based facial recognition (FaceNet, VGG-Face)
- **Libraries**: OpenCV, TensorFlow/PyTorch
